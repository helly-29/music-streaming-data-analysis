# Music Streaming Data Analysis

A complete Python-based data analysis project that explores global music streaming listener preferences using data analysis, statistical operations, preprocessing techniques, and data visualization.


##  Project Overview

This project analyzes listener behavior from a global music streaming dataset.  
It performs:

- Data Cleaning
- Missing Value Detection
- Outlier Detection
- Statistical Analysis
- Data Manipulation
- Data Normalization
- Data Standardization
- Label Encoding
- Data Visualization

The project helps understand:

- User listening habits
- Genre popularity
- Streaming behavior
- Subscription trends
- Engagement patterns


## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy


##  Dataset

Dataset Used:

`Global_Music_Streaming_Listener_Preferences.csv`

Features include:

- Age
- Country
- Top Genre
- Most Played Artist
- Subscription Type
- Minutes Streamed Per Day
- Number of Songs Liked
- Repeat Song Rate
- Listening Time
